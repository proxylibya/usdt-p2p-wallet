# Shared Packages

This directory is reserved for shared packages across the monorepo:

- **@usdt-p2p/common** - Shared types, constants, utilities
- **@usdt-p2p/ui** - Shared UI components
- **@usdt-p2p/validators** - Shared validation schemas

## Structure (Planned)

```text
packages/
├── common/        # Shared types and utilities
├── ui/            # Shared React components
└── validators/    # Shared validation logic
```
